# Derthona Basket Brand Manual

Interactive web edition of the Derthona Basket brand identity manual.

Open `index.html` directly in a browser or serve the repository with any static web server.

## Included

- Brand strategy, tone of voice and manifesto
- Identity foundations and typography system
- Responsive logo usage slider
- Light and dark logo asset previews
- Downloadable official SVG artwork
- Colour palette and respect-area guidance
- Responsive layout and image lightbox
