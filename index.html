05

Visual language

The curve.

A graphic gesture where basketball and territory meet.

The concept

The curve exists where basketball and territory meet. It echoes the trajectory of the ball, the energy of the curva, the hills surrounding Tortona and the vaulted architecture of Pier Luigi Nervi’s Magazzini del Sale. Through repetition and scale, these references become a dynamic and recognizable graphic language.

Install & use the typeface.

01 · Font files

Choose the right format.

Desktop

OTF

Recommended for macOS and Windows. Use it in Word, Adobe apps, Figma and other desktop software.

Desktop alternative

TTF

Use only when an older app or workflow requires TrueType. Install either OTF or TTF, not both.

Web · Primary

WOFF2

The preferred, compressed format for modern browsers. Use it as the first source in CSS.

Web · Fallback

WOFF

Add it after WOFF2 for older browser support. Webfont files do not need to be installed on your computer.

02 · Local installation

Install once.
Then restart your apps.

macOS

  1. Download the OTF file.
  2. Double-click it and select Install Font in Font Book.
  3. Close and reopen any app that was already running.

Windows

  1. Download the OTF file.
  2. Right-click it and choose Install or Install for all users.
  3. Close and reopen any app that was already running.

03 · Alternate style

Activate Stylistic Set 1.

The alternate style is an OpenType feature named ss01. It replaces selected uppercase forms for A, B, D, E, F, H, K, P, Q, R and T. The feature is included in OTF, TTF, WOFF and WOFF2.

Microsoft Word+

Use the desktop app. Select the text, open the Font dialog, choose Advanced, then under OpenType Features set Stylistic sets to 1.

On macOS, the Font dialog may be under Format → Font. Word Online and mobile versions may not expose this control.

Microsoft reference ↗
Adobe Illustrator+

Select the text, then go to Window → Type → OpenType. Open Stylistic Sets and choose Set 1.

Official Illustrator guide ↗
Adobe Photoshop+

Select the text, then open Window → Character. From the Character panel menu choose OpenType, then activate Stylistic Set 1 when available.

If your Photoshop version does not show Stylistic Sets, use Window → Glyphs to select alternates manually.

Official Photoshop guide ↗
Figma+

Select the text layer. In the right sidebar open Typography settings, find OpenType features, then activate Stylistic set 1.

For browser-based Figma, make sure the font is available through the Figma font installer or your organization’s font service.

Official Figma guide ↗
Canva+

Upload the OTF or TTF through Brand Kit if your account and permissions allow custom fonts. Canva does not currently provide a control for OpenType stylistic sets such as ss01.

For alternate letters, prepare the final wording in Illustrator, Photoshop or Figma, convert it to outlines, export SVG or PNG, then upload it to Canva.

Official Canva guide ↗

04 · Web use

Load the font.
Switch on ss01.

@font-face {
  font-family: "Derthona Display";
  src:
    url("DerthonaDisplay-Regular.woff2") format("woff2"),
    url("DerthonaDisplay-Regular.woff") format("woff");
  font-style: normal;
  font-weight: 400;
}

.alternate-style {
  font-feature-settings: "ss01" 1;
}
